Angular Profanity !

A simple client side AngularJS profanity directive based on jQuery.ProfanityFilter jquery plugin.

View on Github »

Example 1

// code: in controller $scope.customSwears = ['ass']; // html: <div class="example box" profanity-filter custom-swears="customSwears"> Often when I go to the bathroom, shit comes out of my ass. </div>
This container will filter the word "ass" but will not filter "shit"
Often when I go to the bathroom, shit comes out of my ass.

Example 2

// code: in controller $scope.customSwears = ['ass']; $scope.externalSwears = 'swearWords.json'; // html: <div id="two" class="example box" profanity-filter external-swears="externalSwears.json" custom-swears="customSwears"> Often when I go to the bathroom, shit comes out of my ass. </div>
This container will filter both the words "shit" and "ass"
Often when I go to the bathroom, shit comes out of my ass.

Example 3

// code: in controller $scope.customSwears = ['ass']; $scope.externalSwears = 'swearWords.json'; $scope.replaceWith = ['fun', 'stuff']; // html: <div id="three" class="example box" profanity-filter external-swears="swearWords.json" custom-swears="customSwears" replace-with="replaceWith"> Often when I go to the bathroom, shit comes out of my ass. </div>
This container will filter both the words "shit" and "ass", but will also change the replacement character from an asterisk (*) a random word listed in the `replaceWith` array.
Often when I go to the bathroom, shit comes out of my ass.

Example 4

// code: in controller $scope.customSwears = ['ass']; $scope.externalSwears = 'swearWords.json'; $scope.replaceWithHash = '#'; // html: <div id="four" class="example box" profanity-filter external-swears="swearWords.json" custom-swears="customSwears" replace-with="replaceWithHash"> Often when I go to the bathroom, shit comes out of my ass. </div>
This container will filter both the words "shit" and "ass", but will also change the replacement character from an asterisk (*) to a pound sign (#)
Often when I go to the bathroom, shit comes out of my ass.

Example 5

// code: in controller $scope.customSwears = ['ass']; $scope.externalSwears = 'swearWords.json'; $scope.replaceWithHash = '#'; $scope.profaneText = function(data, element) { console.log('profane text ',data, element); element.css('color', 'blue'); }; // html: <div id="four" class="example box" profanity-filter external-swears="swearWords.json" custom-swears="customSwears" replace-with="replaceWithHash" filter="false" profane-text="profaneText"> Often when I go to the bathroom, shit comes out of my ass. </div>
This container won't filter anything, but it will do something clever because it contains the word "ass"
Often when I go to the bathroom, shit comes out of my ass.

Example 6

// code: in controller $scope.customSwears = ['ass']; $scope.externalSwears = 'swearWords.json'; $scope.replaceWithHash = '#'; $scope.profaneTextSix = function(data, element) { data.forEach(function (swearWord, index) { var str = '<span class="six-highlight">' + swearWord + '</span>'; element.html(element.html().replace(swearWord, str)); }); }; // html: <div id="four" class="example box" profanity-filter external-swears="swearWords.json" custom-swears="customSwears" replace-with="replaceWithHash" filter="false" profane-text="profaneTextSix"> Often when I go to the bathroom, shit comes out of my ass. </div>
This container won't filter anything, but it will do something clever with the swears that it finds.
Often when I go to the bathroom, shit comes out of my ass.
